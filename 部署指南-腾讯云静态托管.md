三年级语文复习应用 - 使用腾讯云静态托管部署，无需备案，国内访问速度极快。

- ✅ **完全免费**：每月5GB免费流量，足够使用

## 📝 部署步骤

### 第一步：注册腾讯云账号

1. 访问 [腾讯云官网](https://cloud.tencent.com)
2. 点击右上角"注册"按钮
3. 使用手机号或微信完成注册
4. **实名认证**（必须，但不需要备案）：
   - 登录后，点击右上角头像 → "账号信息"
   - 完成个人实名认证（需要身份证照片）
   - 等待审核通过（通常5-10分钟）

---

### 第二步：开通云开发服务

1. 登录腾讯云控制台
2. 搜索并进入 **"云开发 CloudBase"**
   - 直接访问：https://console.cloud.tencent.com/tcb
3. 点击 **"新建环境"**
4. 填写环境信息：
   - **环境名称**：`grade3-chinese`（或其他你喜欢的名称）
   - **套餐版本**：选择 **"免费版"**
   - **地域**：选择离你最近的地域（如"上海"或"广州"）
5. 点击 **"立即开通"**
6. 等待环境初始化完成（约1-3分钟）

---

### 第三步：开启静态网站托管

1. 在云开发控制台，点击左侧菜单 **"静态网站托管"**
2. 点击 **"开始使用"** 或 **"设置"**
3. 开启静态网站托管功能
4. 等待服务开通完成（约30秒）
5. 开通后，你会看到一个 **临时域名**，格式类似：
   ```
   https://xxxx-xxxxx.tcloudbaseapp.com
   ```
   **记住这个域名！这就是你的访问地址，无需备案。**

---

### 第四步：准备上传文件

#### 方法A：使用网页界面上传（最简单）

1. 在"静态网站托管"页面，点击 **"文件管理"**
2. 你会看到一个空的文件列表
3. 准备好要上传的文件：
   - 在你的电脑上打开项目文件夹：
     ```
     /Users/duoji/Documents/GitClone_Projects/三年级上学期期末
     ```
4. **需要上传的文件和文件夹**（全选）：
   ```
   三年级上学期期末/
   ├── index.html          ← 首页（必须）
   ├── css/                ← 样式文件夹
   ├── js/                 ← JavaScript文件夹
   ├── data/               ← 数据文件夹
   ├── pages/              ← 页面文件夹
   └── assets/             ← 资源文件夹（如果有）
   ```

5. **开始上传**：
   - 点击 **"上传文件"** 按钮
   - 选择 `index.html` 文件，上传
   - 点击 **"新建文件夹"**，创建 `css` 文件夹
   - 进入 `css` 文件夹，上传 `css/` 目录下的所有文件
   - 重复以上步骤，上传 `js/`、`data/`、`pages/` 等文件夹

   **注意**：
   - 必须保持原有的目录结构
   - 不要上传 `.git` 文件夹
   - 不要上传 `node_modules` 文件夹（如果有）

---

#### 方法B：使用命令行工具上传（推荐，更快）

如果你熟悉命令行，使用这个方法更快速：

**1. 安装腾讯云CLI工具**

打开终端（Terminal），执行：

```bash
npm install -g @cloudbase/cli
```

如果没有安装npm，先安装Node.js：
- 访问 https://nodejs.org/
- 下载并安装LTS版本

**2. 登录腾讯云**

```bash
cloudbase login
```

- 会打开浏览器，要求你登录腾讯云
- 登录成功后，返回终端

**3. 初始化项目**

在项目目录执行：

```bash
cd "/Users/duoji/Documents/GitClone_Projects/三年级上学期期末"

cloudbase init
```

按提示选择：
- 选择你刚才创建的环境（`grade3-chinese`）
- 选择 "静态网站托管"

**4. 部署项目**

```bash
cloudbase hosting deploy -e grade3-chinese
```

- `-e grade3-chinese` 是你的环境ID
- 等待上传完成（约30秒-2分钟，取决于网速）

**5. 查看部署结果**

```bash
cloudbase hosting list -e grade3-chinese
```

会显示你的临时域名和文件列表。

---

### 第五步：访问你的网站

1. 复制第三步中获得的临时域名：
   ```
   https://xxxx-xxxxx.tcloudbaseapp.com
   ```

2. 在浏览器中打开这个域名

3. 你应该能看到项目首页！

## 🔧 配置选项

### 设置默认首页

1. 在"静态网站托管" → "设置"
2. 找到 **"索引文档"**
3. 设置为 `index.html`
4. 找到 **"错误文档"**
5. 设置为 `index.html`（单页应用推荐）
6. 点击 **"保存"**

### 开启CDN加速（可选）

1. 在"静态网站托管" → "设置"
2. 开启 **"CDN加速"**
3. 等待配置生效（约5-10分钟）
4. 开启后访问速度会进一步提升

## ⚠️ 常见问题

### 1. 上传后访问显示404

**原因**：首页文件名不对或路径错误

**解决**：
- 确保根目录有 `index.html` 文件
- 检查"设置"中的"索引文档"是否设置为 `index.html`

### 2. CSS/JS文件加载失败

**原因**：文件路径不对

**解决**：
- 确保所有文件夹结构与本地一致
- 检查HTML中的文件引用路径是否正确
- 本项目使用相对路径，应该没问题

### 3. 页面能访问，但数据不显示

**原因**：JSON数据文件未上传或路径错误

**解决**：
- 确保 `data/` 文件夹已上传
- 确保所有 `.json` 文件都已上传
- 打开浏览器开发者工具（F12），查看Network标签，看是否有404错误

### 4. 临时域名访问很慢

**解决**：
- 开启CDN加速（见上方"配置选项"）
- 等待CDN配置生效（约5-10分钟）

### 5. 超出免费流量怎么办

**流量消耗预估**：
- 单次访问约消耗：2-5MB（包含所有资源）
- 5GB流量可支持约1000-2500次访问
- 对于个人学习使用，完全够用

**如果超出**：
- 可以升级到付费套餐（很便宜）
- 或者重新创建一个免费环境
